Narrative: About this Data Set

Description

From the Tableau chart on this page, you will be able to learn about the patterns in my WiFi connections and how that translates into where I was geographically. At the beginning of data collection (around Feb 20 to Feb 22), I did not allow the app to run continuously in the background. That is most likely why there are not as many datapoints in that range, since I was on campus and at home a lot, so I should have been connected to WiFi.

More importantly, the visualization you see gives you insight as to where I was in the past week. I plotted the datapoints for each unique BSSID and colorcoded each point by SSID, so that tells you that I might have been walking around as I connect to different access points, but the same network. Moreover, if you look at this past weekend, you'll notice that I was connected to the network named "tree o four" for the majority of the time. This is because, as you might have guessed, I stayed home for the majority of the weekend. But during the weekend, there was a period of time when I was at my friends place (the lone red dot in the plot with SSID "KimchiStew"). By perceiving this plot in this manner, you can get a bit of insight on my life just from looking at a wifi timeline.

Thus, from the above explanations, you can ask and answer questions like: how active was Stephen this weekend?

Design Rationale

I chose to display the information in a timeline format because I wanted the user to visualize how my network connections were changing with respect to the day. Combining this with colorcoding each data entry by SSID, you can clearly see the change in my environment. This display helps the user understand the data through color and time.
